Tool Roster

All FarmMeta manual farming tools

The YAML does not stop at the hoe. There are four separate tool lanes, each with its own crop pool, recipe layout, and 10-tier upgrade track.

Field Crops

Verdant Hoe

Built for wheat, carrots, potatoes, beetroot, nether wart, torchflower crop, and pitcher crop.

  • Recipe: 2 Verdant Ingots, 1 Binding Core, 2 Sticks
  • Starter: Honey Hoe I, 240 durability, $0.07 per item
  • Capstone: Skull Hoe X, x1.80 multiplier, 22% double drop
Wheat Roots Nether Wart

Vertical Harvest

Grove Axe

Targets sugar cane, cactus, and cocoa with a simpler handle layout than the hoe.

  • Recipe: 3 Verdant Ingots, 1 Binding Core, 1 Stick
  • Starter: Honey Axe I, 260 durability, $0.11 per item
  • Capstone: Skull Axe X, x1.80 multiplier, 22% double drop
Sugar Cane Cactus Cocoa

Line Harvest

Harvest Shears

Berry and kelp tool line using extra Crop Fiber directly in the final recipe.

  • Recipe: 2 Crop Fiber, 2 Verdant Ingots, 1 Binding Core
  • Starter: Old Scissors I, 220 durability, $0.07 per item
  • Capstone: Scissors X, x1.78 multiplier, 22% double drop
Sweet Berry Bush Kelp Kelp Plant

Bamboo Lane

Bamboo Blade

Special sword-type farming tool powered by Bamboo Core instead of Binding Core.

  • Recipe: 1 Bamboo Core, 1 Verdant Ingot, 1 Stick
  • Starter: Honey Sword I, 240 durability, $0.11 per item
  • Capstone: Skull Sword X, x1.82 multiplier, 23% double drop
Bamboo Blade Line Bamboo Core
